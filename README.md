# SentinelVision AI

SentinelVision AI is a real-time face detection and intelligent monitoring system developed for security monitoring and identity verification.

It uses computer vision techniques to monitor live camera input, detect human faces, compare detected subjects with registered identities, classify known and unknown users, and log monitoring events through a web-based dashboard.

---

## Features

* Real-Time Camera Monitoring
* Human Face Detection
* Registered Face Database
* Known / Unknown Subject Classification
* Live Monitoring Dashboard
* Activity / Event Logging
* Real-Time Backend Processing
* Frontend Dashboard Integration
